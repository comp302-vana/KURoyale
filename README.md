# KURoyale

A Clash Royale-inspired multiplayer card battle game developed in Java as a 6-person team project for COMP302 at Koç University.

## Features

- Real-time multiplayer card battles over a network
- Custom card deck system
- Game save/load support
- Online connectivity via Cloudflare tunnel relay server

## Tech Stack

- **Language:** Java
- **Build Tool:** Gradle
- **Networking:** Custom relay server with Cloudflare tunnel support

## Getting Started

### Prerequisites

- Java 17 or higher
- Gradle (or use the included wrapper)

### Run

```bash
./gradlew run
```

### Build

```bash
./gradlew build
```
